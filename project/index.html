Projects

Some of the projects I feel are worth mentioning.

  • Harmony

    A tiny wayland compositor written in C.

  • Confy

    The best config manager you'll see today.

  • Ballad

    A command-line argument parser library for C.

  • Daisy

    A lightweight, cross-platform image editor.

  • Vial

    A project generator for Flask.

  • Jstow

    A Go package for using JSON like SQL.

  • Mite

    Webapp for tracking Tachiyomi/Mihon extensions.

  • Moviebox_dl

    A CLI tool for downloading movies directly from your terminal.

  • Monograph

    A webapp for downloading books and research papers.

That's all for now. I'm working on a lot of fun stuff that will hopefully make this list someday.
