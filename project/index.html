Vial

If you use the Flask web framework in Python, you might like this one, I created it with the intention of simplifying the project creation process. Flask does not give any form of official template and as such it is the job of the developer to create the files everytime they wish to work on something new. This process can easily become tiring and repetitive and as such vial offers a way to automate this process. Note: Vial is still very opinionated and the current build style might not fit what you are ordinarily used to. Github
