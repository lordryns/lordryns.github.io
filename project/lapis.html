Features

Lapis is supposed to remove the need for a desktop environment as the shell itself comes with every basic thing one would need on their machine, some include:

  • Dock
  • Wallpaper Daemon with support for icons on the desktop
  • A settings app (In early development)
  • An application launcher
  • Notification bar (To be included)
  • Wifi and Bluetooth support (To be included)
  • Embedded Music player support (To be included)
